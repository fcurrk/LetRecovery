# 无损扩大 C 盘

C 盘满了又不想重装？[工具箱](/guide/toolbox)里的 **无损扩大 C 盘** 可以在**保留数据**的
前提下，把当前系统的 C 盘扩大。

::: warning 仅桌面可用
该功能只在正常 Windows 桌面里规划；真正的磁盘操作会在**重启进 WinPE**后执行（运行中的
系统盘无法在线扩容）。
:::

## 两种扩容方式

打开后填入目标大小，程序会算出两条"天花板"：

| 方式 | 说明 | 风险 |
| --- | --- | --- |
| **方式一：纯扩展** | 目标 = 当前大小 + C 盘**紧邻其后的未分配空间**。不移动任何数据。 | 低（推荐） |
| **方式二：实验性** | 在方式一基础上，进一步**压缩 / 移动**后面的数据分区（如 D:）腾出空间。会移动该分区的数据。 | 高，较慢 |

当你填的目标大小超过"方式一"的天花板时，界面会给出醒目警告，提示这将触发数据移动。

::: tip 目标大小的下限
目标大小不能小于当前大小，且至少要等于**已用空间 + 1 GB**。
:::

## 流程

1. 在桌面打开**工具箱 → 无损扩大 C 盘**，填入目标大小。
2. 程序规划好方案；本机若**没有可用 WinPE**，会先**自动下载** PE。
3. 安装临时 PE 引导项，**重启进入 WinPE**完成实际扩容。
4. 扩容完成后重启回系统，C 盘已变大，数据保留。

::: tip C 盘后面没有空闲空间？
如果 C 盘紧邻其后根本没有未分配空间，可以先用[一键分区](/guide/toolbox#磁盘与分区)
腾出空间，再来扩容；或直接使用"方式二"（注意其数据移动风险）。
:::

::: danger 重要数据请先备份
虽然本功能设计为无损，但任何涉及分区表与数据移动的操作都存在风险。扩容前请备份重要数据，
并确保过程中不断电。
:::
