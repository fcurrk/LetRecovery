# 简易模式（一键重装）

如果你不想关心分区、驱动、无人值守这些细节，**简易模式**（小白模式）把整个重装收敛成
三步：**选系统 → 选版本 → 确认**，常用优化项会自动套用。

## 如何开启

简易模式的开关在**关于**页。打开后主界面会切换到简化的卡片式流程。

::: warning 仅桌面可用
简易模式在 **WinPE 下不可用**——它面向的是"在能正常开机的 Windows 里一键重装系统盘"
这个场景。
:::

## 使用流程

1. 在系统卡片中选择要安装的系统。
2. 选择具体**版本**（如 专业版 / 家庭版）。
3. 确认后开始——其余交给程序。

镜像可以是你本地已有的，也可以配合[在线下载](/guide/online-download)获取。

## 它替你做了什么

简易模式会自动套用一组适合大多数人的设置，省去在[高级选项](/guide/advanced-options)里
逐项勾选：

- **绕过 OOBE 必须联网**（BypassNRO）
- **删除预装 UWP 应用**
- **导入磁盘控制器驱动**（NVMe / AHCI / RAID）并**自动导入**本机驱动
- 系统盘**卷标设为 `OS`**
- **用户名**取自当前登录账户
- 启用**无人值守**、格式化目标盘、**修复引导**、完成后**自动重启**

因为目标通常就是**正在运行的系统盘**，简易模式会走[经 PE 安装](/guide/system-install#安装方式)
流程：先准备 WinPE 引导并重启进入，再在 PE 里格式化、释放镜像、导入驱动、修复引导。

::: danger 仍会格式化系统盘
"简易"指的是操作简单，**不**代表不会清盘。重装系统盘同样会格式化 C:，请先备份重要数据。
:::

## 想要更精细的控制？

需要自定义分区、保留特定驱动、指定自己的 `unattend.xml`、做更多注册表优化时，请关闭
简易模式，改用标准的[系统安装](/guide/system-install) + [高级选项](/guide/advanced-options)。
