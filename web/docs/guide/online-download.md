# 在线下载

**在线下载**页让你不离开 LetRecovery 就能获取资源，分为三个分类标签：

| 标签 | 内容 |
| --- | --- |
| **系统镜像** | 精选、可直接安装的 Windows 镜像。 |
| **软件下载** | 装机后常用的工具。 |
| **显卡驱动** | 常见显卡（GPU）驱动包。 |

## Aria2 加速

下载使用内置的 **Aria2** 引擎：默认多线程（`--split=32`、每服务器最多 16 连接），并开启
**断点续传**（`--continue=true`），比单线程下载更快更稳；中断后重试会从已下载处接着下。

系统镜像下载完成后，切到**系统安装**页，把它选为本地镜像即可。
